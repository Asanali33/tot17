# ⏱️ ТАЙМЕР ДЛЯ ЗАДАЧ - БЫСТРОЕ РУКОВОДСТВО

## ✨ Что было добавлено?

Полнофункциональная система таймера для управления временем выполнения задач с визуальным отслеживанием прогресса.

---

## 🚀 БЫСТРЫЙ СТАРТ (За 1 минуту)

### 1️⃣ Создание задачи с таймером
```
1. Нажмите кнопку добавления задачи
2. Введите название: "Написать отчет"
3. Выберите категорию и дедлайн
4. ⭐ В НОВОМ диалоге установите время: 2ч 30м
5. Задача создана!
```

### 2️⃣ Управление таймером
```
1. Откройте задачу
2. На панели "ТАЙМЕР" нажмите кнопки:
   ▶️  Запустить   - начать отсчет
   ⏸️  Остановить  - приостановить
   🔄 Сбросить    - вернуться к началу
   ✏️  Изменить    - изменить время
```

### 3️⃣ Просмотр статуса
```
На карточке задачи видите:
- ⏱ Время выполнения: 2ч 30м
- ⏱ Таймер: 2м 15с (только во время работы)
```

## 🎯 Основные операции

| Операция | Где | Как |
|----------|------|-----|
| **Установить время** | При добавлении | Диалог с 3 полями ввода (ч:м:с) |
| **Запустить** | Экран деталей | Кнопка ▶ или функция `startTimer()` |
| **Остановить** | Экран деталей | Кнопка ⏸ или функция `stopTimer()` |
| **Сбросить** | Экран деталей | Кнопка 🔄 или функция `resetTimer()` |
| **Изменить** | Экран деталей | Кнопка ✏ или функция `setTaskDuration()` |

## 💾 Где сохраняется время?

```dart
Task {
  title: "Написать отчет"
  estimatedDuration: Duration(hours: 2, minutes: 30)  ← Установленное время
  timerStartedAt: DateTime(2024, 04, 15, 14, 20)      ← Когда запустили
  isTimerActive: true                                   ← Активен ли сейчас
}
```

## 🔧 Для разработчиков

### Основные методы в TaskService

```dart
// установить время
taskService.setTaskDuration(0, Duration(hours: 2, minutes: 30));

// запустить/остановить
taskService.startTimer(0);
taskService.stopTimer(0);

// получить информацию
Duration? remaining = taskService.getRemainingTime(0);
double progress = taskService.getTimerProgress(0);
bool expired = taskService.isTimeExpired(0);

// форматированные строки
String formatted = taskService.getFormattedRemainingTime(0);
```

## ❓ Часто задаваемые вопросы

**Q: Потеряется ли время при перезагрузке приложения?**
> Да, таймер сбросится. Это сделано преднамеренно для простоты.

**Q: Могу ли я изменить время во время работы таймера?**
> Да, кнопка "Изменить" всегда доступна.

**Q: Что происходит при истечении времени?**
> Индикатор становится красным, таймер останавливается автоматически.

**Q: Могу ли я создать задачу без времени?**
> Да, время опционально. Пропустите диалог выбора.

**Q: Сколько задач могут иметь активные таймеры?**
> Неограниченно! Все таймеры работают независимо.
