🚀 КАК ЗАПУСТИТЬ БЭКЕНД:

ВАРИАНТ 1 - Простой способ (Windows):
- Откройте: backend/start_backend.bat
- Двойной клик!

ВАРИАНТ 2 - Расширенный способ:
- Откройте: backend/service_manager.bat
- Выберите опции в меню

ВАРИАНТ 3 - Консоль:
1. cd backend
2. npm install (первый раз)
3. node server.js

=====================================

✅ ПРОВЕРКА:

Когда бэкенд запущен, вы должны увидеть:
- "Connected to MongoDB"
- "Server running on port 3000"

Затем:
- Откройте эмулятор Android (или iOS)
- Попробуйте регистрацию
- Должно работать! ✅

=====================================

📋 ТРЕБОВАНИЯ:

- Node.js (скачать с nodejs.org)
- MongoDB (локальная или Atlas облачная)
- Android/iOS эмулятор или физическое устройство

=====================================

❓ ЧТО ДЕЛАТЬ ЕСЛИ ВСЕ РАВНО НЕ РАБОТАЕТ:

1. Проверьте MongoDB:
   - Windows: Services -> MongoDB должен быть запущен
   - Linux/Mac: ps aux | grep mongo

2. Проверьте порт 3000:
   - netstat -ano | findstr :3000
   - Если занят: service_manager.bat -> Опция 5

3. Проверьте сетевое соединение:
   - Убедитесь что эмулятор может подключаться к хосту
   - Для физического устройства: используйте IP вашей машины (ipconfig)

4. Прочитайте START_BACKEND_GUIDE.md для подробных инструкций
