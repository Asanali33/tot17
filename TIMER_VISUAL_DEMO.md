## 4️⃣ Состояния таймера

### Таймер остановлен
```
    ┬────────┬
    │        │
    │ 02:30:00 ← Полное время
    │        │
    ├────────┤
    │◉ Серый │  ← Без прогресса
    │ 0%     │
    └────────┘
```

### Таймер активен (1 мин прошло)
```
    ┬────────┬
    │        │
    │ 02:29:00 ← Оставшееся время
    │        │
    ├────────┤
    │█ Синий │  ← ~1% прогресса
    │ 1%     │
    └────────┘
```

### Таймер завершен
```
    ┬────────┬
    │        │
    │ 00:00:00 ← Время истекло
    │        │
    ├────────┤
    │ Красный│  ← 100% прогресса
    │ 100%   │
    │⏰⏰⏰⏰⏰│  ← Звуковое уведомление
    └────────┘
```
