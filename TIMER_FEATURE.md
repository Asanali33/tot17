# Функция Таймера для Задач

## 📋 Описание
Добавлена полнофункциональная система таймера для управления временем выполнения задач. Каждая задача теперь может иметь установленное время выполнения с активным отслеживанием прогресса.

## ✨ Основные возможности

### 1. Установка времени выполнения
- При **добавлении новой задачи** пользователю предлагается установить время выполнения (часы, минуты, секунды)
- Время можно установить позже через экран деталей задачи
- Формат: ЧЧ:ММ:СС

### 2. Управление таймером
- **Запустить** - начать отсчет времени
- **Остановить** - приостановить таймер
- **Сбросить** - вернуть таймер к начальному времени
- **Изменить время** - отредактировать время выполнения

### 3. Визуальное отображение
- **Круговой индикатор прогресса** - показывает процент выполненного времени
- **Таймер обратного отсчета** - отображает оставшееся время (ЧЧ:ММ:СС)
- **Цветовая индикация** - синий цвет при активном таймере, красный когда время истекло
- **Значок таймера** на карточке задачи с текущим временем

## 🔧 Технические изменения

### Модель Task (lib/models/task.dart)
```dart
Duration? estimatedDuration;      // Установленное время выполнения
DateTime? timerStartedAt;         // Время начала таймера
bool isTimerActive;               // Активен ли таймер
```

### Сервис TaskService (lib/services/task_service.dart)
Добавлены методы:
- `setTaskDuration(int index, Duration duration)` - установить время
- `startTimer(int index)` - запустить таймер
- `stopTimer(int index)` - остановить таймер
- `resetTimer(int index)` - сбросить таймер
- `getRemainingTime(int index)` - получить оставшееся время
- `getTimerProgress(int index)` - получить прогресс (0-1)
- `isTimeExpired(int index)` - проверить, истекло ли время
- `getFormattedDuration(int index)` - получить отформатированное строку
- `getFormattedRemainingTime(int index)` - получить отформатированное оставшееся время

### Экраны

#### HomeScreen (lib/screens/home_screen.dart)
- Добавлен диалог для установки времени при создании задачи
- Интерфейс: три поля для ввода (часы, минуты, секунды)
- Отображение итогового времени в формате ЧЧ:ММ:СС

#### TaskDetailScreen (lib/screens/task_detail_screen.dart)
- Полная панель управления таймером с:
  - Круговым индикатором прогресса
  - Обратным отсчетом времени
  - Кнопками управления (Запустить, Остановить, Сбросить)
  - Кнопкой редактирования времени

#### TaskTile (lib/widgets/task_tile.dart)
- Отображение времени выполнения на карточке задачи
- Иконка таймера с соответствующим цветом (тертичный)
- Показ оставшегося времени при активном таймере

## 📱 Интерфейс пользователя

### Добавление задачи с таймером
1. Введите название задачи
2. Выберите категорию
3. Установите дедлайн (опционально)
4. **В новом диалоге установите время выполнения** ← Новой функция!
5. Задача будет создана с установленным временем

### Управление таймером задачи
1. Нажмите на задачу или перейдите в детальный вид
2. В разделе "Таймер" установите или отредактируйте время
3. Используйте кнопки:
   - ▶️ **Запустить** - начать отсчет
   - ⏸️ **Остановить** - приостановить
   - 🔄 **Сбросить** - вернуться к начальному времени
   - ✏️ **Изменить время** - отредактировать длительность

## 📊 Информация о таймере на карточке
На каждой карточке задачи отображается:
- "Время выполнения: ЧЧМ СС" - установленное время (например: "1ч 30м 45с")
- "Таймер: ММ СС" - оставшееся время (только когда таймер активен)

## ⏰ Особенности
- Таймер обновляется в реальном времени
- При истечении времени таймер автоматически останавливается
- Таймер состояние сохраняется при перемещении между экранами
- Можно изменить время даже во время активного таймера
- История изменений отслеживает все изменения времени выполнения

## 🚀 Как использовать

### Пример использования
```
1. Создайте новую задачу "Написать отчет"
2. При добавлении установите время: 2 часа 30 минут
3. Откройте эту задачу
4. Нажмите "Запустить" для начала таймера
5. Таймер будет отсчитывать время
6. По окончании вы увидите сообщение "⏰ Время истекло!"
```

## 🔔 Уведомления
При истечении времени:
- Индикатор прогресса становится красным
- Таймер останавливается автоматически
- На карточке задачи отображается "⏰ Время истекло!"

## 📝 Примечания
- Время выполнения опционально - вы можете создавать задачи без установки времени
- Таймер не влияет на статус задачи (к выполнению, в работе, выполнено)
- Все данные таймера хранятся в памяти приложения
- При перезагрузке приложения активный таймер будет сброшен
